Expression
	A meaningful combination of values and operators that EVALUATES TO A SINGLE VALUE.  
		Operator
		Operand
		Binary/Unary
	
			Calling a function is a valid expression.
			3+b+c+d
			If 'Sunny' && haveBike || ...
			str.length !==0
			true


	NOT EXPRESSIONS - these are STATEMENTS
		Declaring a variable.
		IF ELSE
		FOR LOOPS
		WHILE LOOPS
